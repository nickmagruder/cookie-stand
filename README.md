# Salmon Cookies Website

## CodeFellows Code 201 Salmon Cookies Project
This is a 2-page website built during the CodeFellows 201 project. The page is meant to represent a public-facing website for a chain of cookie shops. The sales.html page is meant to represent a back end page for managers to estimate cookie sales from current or future stores.
